## TravelB — это полнофункциональная система для планирования и управления путешествиями. Платформа позволяет пользователям:

## Функционал

Регистрация и авторизация пользователей

Личный кабинет с персональной информацией

Просмотр доступных статей

Создание и редактирование статей


## Технологии

Backend: **Django** 

Frontend: **HTML, CSS**

База данных: **SQLite** 

## Запуск
````
python -m venv .venv
````
````
.venv\Scripts\activate
````
````
pip install -r requirements.txt
````
````
python manage.py migrate
````
````
python manage.py loaddata perfect_backup.json
````
````
python manage.py runserver
````
